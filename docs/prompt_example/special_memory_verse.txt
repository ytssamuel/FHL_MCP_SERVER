請幫我做背經輔助（記憶技巧與計劃）。

請按照以下步驟進行：

選節

使用 get_bible_verse 取得目標經節（可多節）。

分段與節奏

句法切分（自然停頓）→ 分段記憶。
關鍵詞掛鉤（圖像/首字/押韻）。

複習計劃

艾賓浩斯間隔：Day 0、1、3、7、14、30。
每次複習：覆誦→遮詞→默寫→檢核。

理解支撐

提供背景與主旨（2-3 句）
列出 2-3 處交叉經文（search_bible）

工具（可選）

多版本對照（translation compare）
原文關鍵詞（get_word_analysis + lookup_strongs）

輸出格式

目標經節
切分與記憶鉤子
複習日程
背景與交叉
檢核清單
