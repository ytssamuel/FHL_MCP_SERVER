請幫我深入研讀一節經文（專業解經分析）。

請按照以下步驟進行研讀：

經文內容

使用 get_bible_verse 取得指定經文（unv 版本）。
同時獲取包含 Strong's Number 的版本以便原文分析。

原文字彙分析

使用 get_word_analysis 取得該節希伯來文/希臘文分析。
列出重要字詞的原文、詞性、字形變化與語法功能。

關鍵字詞研究

針對關鍵字使用 lookup_strongs 查詢 Strong 字典。
解釋原文意義、語域、神學含義，並列出同源字與使用。
如需統計出現：使用 search_bible（search_type=hebrew/greek）。

註釋與解經

使用 get_commentary 查詢該節註釋。
綜合不同觀點，提出解經要點與應用建議。

相關經文連結

使用 search_bible 搜尋相關主題或關鍵字。
列出 3-5 處相關經文供交叉參考。

研讀總結

核心信息
神學意義
實際應用
思考問題
請以結構化、易讀的方式呈現結果。
