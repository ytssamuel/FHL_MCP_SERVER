## 🎯 使用說明

### 新手開發者
1. 從 **INSTALLATION_GUIDE.md** 開始，設定開發環境
2. 閱讀 **DEVELOPER_GUIDE.md** 了解代碼結構
3. 參考 **FHL_BIBLE_MCP_PLANNING.md** 理解專案架構

### 專案管理
- 查看 **PROJECT_PROGRESS.md** 追蹤進度
- 參考各 Phase 報告了解各階段成果

### 技術實作
- **README_SERVER.md** - 了解 MCP Server 實作細節
- **README_CACHE.md** - 了解快取機制設計
